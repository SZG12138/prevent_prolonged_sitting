# 防久坐提醒 – 一个无法忽视的全屏遮罩提醒工具

## 这是什么？

一个极简的 Windows 桌面工具，到点**用全屏半透明遮罩强制提醒你起身活动**。  
它常驻系统托盘，无日志、无缓存、不写注册表，来去无痕。

## 核心功能

- 默认每 **45 分钟** 弹出全屏遮罩提醒，无法忽视（但可 `Esc` 关闭或点击按钮）
- 支持 **自定义倒数计时**（例如：坐下 30 分钟后打开，只需设 15 分钟后提醒）
- 临时想再坐一会？点击「再坐 5 分钟」稍后提醒
- 右键托盘菜单可随时**重置为默认 45 分钟**，重新开始完整计时
- 建议在起身离开座位后手动重置，更贴合真实使用场景
- **纯绿色**：打包后为单个 exe，不生成任何缓存、日志或系统文件

## 界面预览

- 全屏半透明黑色遮罩 + 白色大字提醒
- 两个按钮：**我已起身**（重置并继续保护） ｜ **再坐 5 分钟**
- 系统托盘图标，右键菜单：**自定义倒计时**、**重置计时（45分钟）**、**退出**

## 快速上手

### 方式一：直接下载 exe（推荐多数用户）

前往 [Releases 页面](../../releases) 下载最新版 `防久坐提醒.exe`，双击运行即可。  
程序会出现在系统托盘区（任务栏右下角），右键图标可以自定义倒计时或退出。

### 方式二：从源码运行

需要 Python 3.x 和以下依赖：
pip install -r requirements.txt

然后运行：
python src/app.py

自己打包成 exe：
pip install pyinstaller

pyinstaller --onefile --windowed --name "防久坐提醒" src/app.py

生成的 exe 位于 dist/ 目录。

### 设置开机自启动
程序目前不内置自启开关，但你可以通过 Windows 自带的启动文件夹轻松实现：
找到 防久坐提醒.exe（或它的快捷方式）
按 Win + R 打开运行窗口，输入 shell:startup 回车
将 exe 的快捷方式拖进这个「启动」文件夹即可
下次开机时，程序会自动在后台运行
取消自启：再次打开 shell:startup，删除该快捷方式
